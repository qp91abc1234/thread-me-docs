### 项目名称：

**英文名称**：ThreadMe
**中文名称**：线索社交

### 技术栈：

* **后端**：NestJS（支持高效的 RESTful API 和 WebSocket 服务）
* **后台管理**：Vue3 + Element Plus + Pinia + Vue Router
* **前端**：UniApp（目标平台为微信小程序）

### 功能模块：

1. **用户管理**：

   * **注册与登录**：

     * 支持邮箱、手机号注册，微信扫码登录
     * 忘记密码功能
     * 登录后生成 JWT Token 进行身份验证
   * **个人资料**：

     * 用户可以设置个人头像、昵称、个性签名
     * 用户能够编辑和查看个人信息
     * 支持上传头像（支持图片格式）
   * **账户安全**：

     * 密码修改和二次验证（如邮箱验证、短信验证码）

2. **帖子管理**：

   * **帖子发布与编辑**：

     * 用户可发布文本、图片、视频等多媒体内容
     * 支持内容编辑、排版、插入链接
   * **帖子交互**：

     * 用户可以对帖子进行点赞、评论、分享操作
     * 支持评论回复，帖子评论数实时更新
   * **帖子分类与筛选**：

     * 通过标签或分类对帖子进行归类
     * 根据热门程度、最新发布等方式筛选帖子

3. **消息通知系统**：

   * **通知类型**：

     * 系统通知（如活动提醒、公告等）
     * 用户互动通知（如新评论、新点赞等）
   * **实时消息**：

     * 后期可实现实时聊天功能，基于 WebSocket 技术
   * **消息管理**：

     * 用户可查看未读消息，并标记已读
     * 消息可以通过推送发送给用户

4. **社交互动**：

   * **关注与粉丝功能**：

     * 用户可以关注其他用户，建立社交网络
     * 支持查看自己的关注列表与粉丝列表
   * **分享功能**：

     * 支持将帖子分享到其他平台（如微信、朋友圈等）
     * 实现社交平台之间的互通

5. **后台管理系统**：

   * **用户管理**：

     * 管理员可以查看、编辑用户信息，进行用户封禁操作
   * **帖子管理**：

     * 管理员可以审核、删除违规帖子
     * 帖子分类管理
   * **内容审核与监控**：

     * 配置关键词过滤、图片和视频内容审核
     * 系统自动识别、报告不当内容
   * **消息管理**：

     * 管理员可以发送公告或通知
     * 用户消息的查看和管理

6. **文件上传与存储**：

   * 使用阿里云 OSS 或本地存储进行文件上传
   * 图片、视频等多媒体内容上传，进行文件大小和格式校验
   * 上传后生成文件的访问 URL，用于前端展示

7. **搜索与推荐功能**：

   * **搜索**：

     * 用户可以根据关键词搜索帖子、用户、话题等
     * 支持热门搜索词推荐
   * **内容推荐**：

     * 基于用户行为（点赞、评论、关注）推荐相关内容
     * 推荐算法可以根据用户兴趣进行调整

### 技术实现：

* **后端（NestJS）**：

  * **JWT 认证**：用于保护 API 接口，确保只有合法用户可以访问
  * **WebSocket**：后期可使用 WebSocket 实现实时消息和实时通知
  * **数据库**：使用 PostgreSQL 或 MySQL 作为关系型数据库存储用户、帖子、评论等数据
  * **缓存与性能优化**：使用 Redis 缓存热点数据，如用户信息、帖子列表等，提升系统性能
  * **文件存储**：采用阿里云 OSS 存储图片、视频等文件，确保高可用性和可靠性

* **前端（UniApp）**：

  * **多平台支持**：UniApp 支持将应用打包成微信小程序
  * **页面与组件**：使用 Vue3 组件化开发，结合 Pinia 进行状态管理，Vue Router 实现页面跳转
  * **UI 设计**：基于 Element Plus 设计后台管理系统，前端页面采用简洁现代的设计风格
  * **小程序适配**：根据微信小程序的要求，进行相应的适配，处理小程序的限制和特性

* **后台管理（Vue3 + Element Plus）**：

  * **动态数据展示**：展示用户、帖子、评论等实时数据，结合图表等方式进行统计
  * **权限控制**：后台管理员通过角色权限控制不同的操作权限
  * **页面路由**：使用 Vue Router 实现灵活的路由配置
  * **表单管理**：Element Plus 提供表单组件，支持用户信息、帖子等数据的创建和编辑

### 部署与上线：

* **后端部署**：

  * 使用 Docker 容器化部署，方便横向扩展与负载均衡
  * 云服务器（如阿里云、AWS）部署后端服务，保证高可用性
* **前端部署**：

  * UniApp 打包成微信小程序，提交审核后上线
  * 后期可扩展至其他平台，如 iOS、Android 等
* **CI/CD 管道**：

  * 使用 GitHub Actions 或 GitLab CI 实现自动化部署
  * 提供自动化测试、构建与部署流程

### 安全与性能：

* **数据加密**：

  * 所有敏感数据（如密码、支付信息）进行加密存储
  * HTTPS 加密通讯，防止数据在传输过程中被窃取
* **高可用架构**：

  * 通过负载均衡与数据库分片实现系统的高可用与高并发处理能力
* **API 限流与防护**：

  * 设置 API 调用频率限制，防止恶意请求或爬虫攻击
