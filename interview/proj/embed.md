lyric 组件：
    解析 lyric 文件
    根据播放时间计算当前激活索引
    根据激活索引计算滚动距离并进行滚动
    
emoji 功能    
    插入 emoji：
        在 emoji 面板选择 emoji
        输入框根据插入位置插入 emoji 文本，ex.'[哭]'
    解析评论内容：通过正则将 emoji 文本替换为 img 标签
    显示评论内容：通过 v-html 进行显示

评论模块
    支持主评论和子评论的二级结构，包含发表、展示、点赞、删除等基础功能

虚拟列表
    用 transform: translateY() 模拟滚动
    定高虚拟列表
        根据滚动距离 + 每项高度 -> 起始索引
        根据起始索引 + 视口高度 + 每项高度 -> 结束索引
        根据起始索引 + 结束索引 -> 偏移距离 + 渲染数据
        （偏移距离：起始项顶部到列表顶部的距离）
    不定高虚拟列表
        预设每项高度，计算列表项位置信息
        根据滚动距离 + 列表项位置信息 -> 起始索引
        根据起始索引 + 列表项位置信息 + 视口高度 -> 结束索引
        根据起始索引 + 结束索引 -> 偏移距离 + 渲染数据
        渲染结束后更新列表项位置信息
        
瀑布流组件
    后端返回图片宽高
    用 transform 进行位置摆放
    非虚拟
        根据数据列表中的图片尺寸进行位置计算
        渲染时根据计算好的位置进行摆放
    虚拟
        在非虚拟的基础之上根据滚动距离和视口高度对渲染数据进行过滤
